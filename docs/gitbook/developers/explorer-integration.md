# Block Explorer Integration

Build block explorers using Archivas API.

## Endpoints for Explorers

- `GET /blocks/recent?limit=100` - List blocks
- `GET /block/<height>` - Block details
- `GET /tx/recent?limit=200` - Recent transactions
- `GET /account/<address>` - Account info

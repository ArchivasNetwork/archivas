# How to Contribute

Join the Archivas development community!

## Ways to Contribute

### Code
- Fix bugs
- Add features
- Improve performance
- Write tests

### Documentation
- Fix typos
- Add examples
- Write guides
- Translate

### Testing
- Test new features
- Report bugs
- Suggest improvements
- Run testnet nodes

## Getting Started

1. Fork https://github.com/ArchivasNetwork/archivas
2. Create branch: `git checkout -b feature/my-feature`
3. Make changes
4. Test locally
5. Submit pull request

## Development Setup

```bash
git clone https://github.com/YOUR_USERNAME/archivas.git
cd archivas
go mod download
go build ./...
go test ./...
```

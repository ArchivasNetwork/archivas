## Quick Reference

**Base URL:** `https://seed.archivas.ai`

**Key Endpoints:**
- `GET /account/<address>`
- `GET /chainTip`
- `GET /blocks/recent?limit=N`
- `GET /block/<height>`
- `GET /tx/recent?limit=N`
- `POST /submit`

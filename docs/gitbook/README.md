Archivas is a Layer 1 blockchain that uses **Proof-of-Space-and-Time** (PoST) consensus - the same mechanism pioneered by Chia Network. Instead of wasting energy (PoW) or requiring capital lockup (PoS), Archivas secures its network with:

* **Disk Space (Proof-of-Space)** - Farmers allocate storage to create "plots"
* **Sequential Time (VDF)** - Timelords compute Verifiable Delay Functions

**Native Token:** RCHV\
**Block Reward:** 20 RCHV\
**Block Time:** \~20-30 seconds\
**Network:** archivas-devnet-v4 (public testnet)

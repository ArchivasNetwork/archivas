# Cryptography

Cryptographic algorithms used in Archivas.

## Signatures
- **Ed25519:** Public key crypto
- **Blake2b:** Hashing

## Addresses
- **Bech32:** Encoding format
- **Blake2b-160:** Address derivation

## Transactions
- **RFC 8785:** Canonical JSON
- **Blake2b-256:** Transaction hashing
- Domain separation: `Archivas-TxV1`

Full spec: [Wallet Primitives](../../wallet-primitives.md)
